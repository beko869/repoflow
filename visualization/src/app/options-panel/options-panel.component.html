Help for selecting quality metric and files

Selecting a file from the 'Add file to visualization' dropdown visualizes the file in the trend area and adds it to the file list below the dropdown.

The tab 'Compare file versions' allows you to compare file versions by clicking on a file node in the trend area. After that you have to click 'Fixate revision of first file' and choose a second file in the trend area. Clicking 'Show File Difference View' opens a diff view of the two selected file versions.
